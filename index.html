Generative Audio Lands

Generative Audio Lands is an upcoming interactive audio experiment. It captures audio input streams (via mic or system player) and converts sound frequencies to deform 3D plane dimensions in real-time.

Planned Architecture

  • Web Audio API analyser node mapping amplitude to vertex positions.
  • Simplex noise terrain generation for natural wave motion and valleys.
  • Custom bloom filters rendering neon wireframe lines that bounce to the beat.
  • Audio compression controls enabling smooth data transfer ratios.
Web Audio API Three.js Simplex Noise GLSL Bloom
